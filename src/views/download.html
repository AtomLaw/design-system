{{title}}

You can download the latest pre-release here on Github. We'd suggest following the outlined options below.

To use Design System, copy/download the file found on the master branch to your website. This file contains all current components and provides support for any needed icons, typography, or styling. We recommend including it in the HEAD html tag for use throughout your website/application. {{escape ''}}

By default, button colors are set to $jdrf-green. However the button colors should vary by program. It is recommended to change button colors as soon as the Design System file has been placed in the HEAD html tag.

View our Colors section for an example of how to overwrite these button colors for your project.

Currently, Design System only supports a few components - none of which require JavaScript. If font files are needed for the website/application, copy/download the font files found in the master branch It's recommended to place the font folder in a similar path to the CSS files. {{escape ' └── ./ ├── css/ └── design-system.css └── fonts/ └── gotham/ '}} If icons are a requirement, Design System already includes all necessary styles, but requires the font file from Google's CDN. Include this in the `HEAD` html tag right before the Design System css file. {{escape ' '}}
